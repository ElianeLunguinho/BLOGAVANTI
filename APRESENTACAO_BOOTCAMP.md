# Blog Avanti - Resumo para Apresentação

## O que é o projeto?
Plataforma de troca de conhecimentos onde pessoas podem compartilhar suas habilidades e aprender com outros.

## Tecnologias Usadas
- **React** - Framework JavaScript
- **Vite** - Build tool
- **Framer Motion** - Animações
- **React Router** - Navegação
- **React Icons** - Ícones
- **CSS** - Estilização

## Pastas Principais para Apresentar o Código

| Pasta | Descrição |
|-------|-----------|
| `src/components/` | Componentes React (Hero, Footer, Navbar, Form, Modal, Cards) |
| `src/pages/` | Páginas (Home, KnowledgeList, Register) |
| `src/context/` | Contextos (ThemeContext - tema claro/escuro, AppContext - estado) |
| `src/data/` | Dados de exemplo |

## O que foi desenvolvido
- Homepage com carrossel de imagens animado
- Página de listagem de conhecimentos
- Página de cadastro de ofertas
- Sistema de tema claro/escuro
- Animações com Framer Motion
- Formulários interativos
- Modal para editar/excluir
- Toast notifications

## Como executar o projeto
```
bash
npm install
npm run dev
```
