# 🚀 Blog Avanti
## Apresentação do Projeto

---

## 📌 O que é o Blog Avanti?

Plataforma web de **troca de conhecimentos** onde pessoas podem:
- Compartilhar suas habilidades
- Conectar-se com outras pessoas
- Trocar oportunidades de aprendizado

---

## 🎯 Problema

- Pessoas com conhecimentos valiosos não têm onde compartilhar
- Difficultade em encontrar oportunidades de aprendizado
- Falta de conexão entre quem quer aprender e quem quer ensinar

---

## 💡 Solução

Uma plataforma que conecta pessoas dispostas a:
- **Compartilhar** conhecimento
- **Aprender** novas habilidades
- **Trocar** experiências

---

## ✨ Funcionalidades Principais

| 功能 | Descrição |
|------|------------|
| 🏠 Página Inicial | Seção Hero com proposta e chamada para ação |
| 📚 Lista de Conhecimentos | Grid de cards com todas as ofertas |
| 📝 Cadastro | Registro de usuários na plataforma |
| ➕ Criar Ofertas | Criar e gerenciar ofertas educacionais |
| ⚙️ Gerenciamento | Editar e remover ofertas |
| 🌙 Modo Escuro | Alternância entre tema claro e escuro |
